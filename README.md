# .dotfiles

## Installation

```bash
$ git clone https://github.com/kiyodori/dotfiles.git ~/.dotfiles
$ cd .dotfiles
$ git submodule init
$ git submodule update
$ chmod +x bootstrap.sh
$ source bootstrap.sh
```

### peco

For zsh filtering, install peco.

##### Mac OS X / Homebrew

```bash
brew tap peco/peco
brew install peco
```
